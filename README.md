# Personal Website

A simple and clean personal portfolio built with [Jekyll](https://jekyllrb.com/).

## About

This is my personal website showcasing my work, projects, and interests. Built using the al-folio Jekyll theme for a clean, responsive design with light/dark mode support.

## Features

- Clean, responsive design
- Light/dark mode toggle
- Projects showcase
- Research/Publications
- CV/Resume
- Blog posts with math support
